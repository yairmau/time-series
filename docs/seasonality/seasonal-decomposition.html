20.1 decompose data

seasonal_decompose returns an object with four components:

  • observed: \(Y(t)\)
  • trend: \(T(t)\)
  • seasonal: \(S(t)\)
  • resid: \(e(t)\)

Additive model: \[ Y(t) = T(t) + S(t) + e(t) \]

Multiplicative model: \[ Y(t) = T(t) \times S(t) \times e(t) \]
